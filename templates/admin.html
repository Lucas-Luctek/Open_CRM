Droits des personnes concernées

Droit d'accès & export

Bouton "Export RGPD" sur chaque fiche prospect → télécharge toutes les données en JSON.

Droit à l'effacement

Bouton "Supprimer" sur la fiche prospect → supprime le prospect + toutes ses données liées.

Droit d'opposition (opt-out)

Case "Ne plus contacter" dans l'édition du prospect → bloque tous les emails.

Droit de rectification

Bouton "Éditer" sur chaque fiche → modification de toutes les données.
