# Contribuer au CRM Open Source

Merci de l'intérêt pour le projet ! Voici comment participer.

---

## Signaler un bug

1. Vérifiez que le bug n'est pas déjà signalé dans les [Issues](https://github.com/Lucas-Luctek/Open_CRM/issues)
2. Ouvrez une nouvelle Issue en utilisant le template **Bug report**
3. Décrivez clairement : version utilisée, OS, étapes pour reproduire, comportement attendu vs observé

---

## Proposer une fonctionnalité

Ouvrez une Issue avec le template **Feature request** en expliquant le besoin et le cas d'usage concret.

---

## Contribuer du code

1. Forkez le dépôt
2. Créez une branche : `git checkout -b ma-fonctionnalite`
3. Faites vos modifications (une fonctionnalité par branche)
4. Testez que l'application démarre et fonctionne correctement
5. Ouvrez une Pull Request en décrivant ce que vous avez changé et pourquoi

### Règles de base

- Respectez le style du code existant (Python/Flask)
- Ne modifiez pas `crm.db`, `uploads/`, `.env` — ces fichiers sont locaux
- Les PRs modifiant `app.py` doivent être testées sur une installation fraîche
